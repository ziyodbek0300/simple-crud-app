# CRUD API
## Description

### Set environment

```bash
git clone https://github.com/SpawnLeon/simple-crud-api
cd simple-crud-api
git task-3-simple-crud-api
npm install
```
Rename **.env.example** file to **.env**.

### Run application
Development mode:
```bash
npm run start:dev
```

Production mode:
```bash
npm run start:prod
```
